Sobre

Este projeto consiste na criação de um site para uma Pizzaria fícticia. Para o desenvolvimento desse projeto foram necessárias diversas habilidades com CSS, HTML e JS. Nele existem questões referentes a posicionamento de elementos, display grid, flex, hover entre outros. Cada sessão do site consta no menu superior e ao clicar em cada parte o úsuario é navegado para a sessão ou área requisitada (com o click).
Este é um projeto responsivo, onde o javascript auxiliou para a criação de um menu mobbile (hamburguer) para telas menores e o display grid para o posicionamento do cardápio.
No entanto, o objectivo principal é demonstrar a responsividade da página com mais leveza e interação social.
Att.: No momento não se encontram habilitadas as funções dos botões.
